# Домашнее задание 10

## Описание
Контейнер генерирует тестовые данные (как в слайде 25), затем выполняет  
**anti left**, **anti right**, **anti outer join** с помощью `dplyr`.

Результаты сохраняются в директорию `data`.

При старте контейнера:
- выводится версия `dplyr`
- запускается обработка данных
- входные файлы читаются из смонтированной папки `data`

## Структура проекта
```
.
├── Dockerfile
├── run.R
└── data/
```

## Сборка контейнера
```
docker build -t dplyr-anti-joins .
```

## Запуск контейнера
```
docker run -v $(pwd)/data:/app/data dplyr-anti-joins
```

После запуска внутри `data` появятся:
- anti_left.csv  
- anti_right.csv  
- anti_outer.csv
